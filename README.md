# Real Estate Twilight AI Tool

## Description
This is a custom AI-powered internal tool that converts daytime real estate photos into realistic twilight images while preserving all architectural structures. Designed for magazine-quality outputs.

## Features
- Automatic image segmentation (sky, windows, landscaping, building)
- Twilight sky generation with SDXL and ControlNet
- Color grading & tone mapping
- Window and exterior lighting simulation
- Fully automatic with optional user adjustments
- Cloud GPU ready
